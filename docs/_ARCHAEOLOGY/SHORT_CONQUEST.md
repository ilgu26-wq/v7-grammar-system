## 4. 전반/후반 BULL 분리

### 🔹 전반 BULL (방출 성공 구간)

**특징:**
- 에너지 충분
- 실패 = 즉시 역행

**판단 로직:**
```
MAE @ Bar 3 < 4pt → HOLD
MAE @ Bar 3 ≥ 4pt → CUT
```

**결과:**
| 지표 | 값 |
|:-----|:---|
| WR | 90.9% |
| EV | +8.2pt |
| N | 11 |

👉 **절대값이 유효**

### 🔹 후반 BULL (흡수 전환 구간)

**특징:**
- 에너지 소진 직전
- 실패 = 천천히 흡수

**실험 결과:**
| 축 | 유효성 | 분리력 |
|:---|:-------|-------:|
| MAE 절대값 | ❌ | - |
| FR 변화율 | ❌ | 0.31 |
| **ΔMAE** | ✅ | **0.83** |

**판단 로직:**
```
ΔMAE < 2pt → HOLD
ΔMAE ≥ 2pt → CUT
```

**결과:**
| 지표 | Before | After |
|:-----|-------:|------:|
| WR | 37.5% | 50.0% |
| EV | -2.5pt | 0pt |

👉 **지는 구간을 "안 지게" 만듦**

## 5. EV 분석 통합 결론

### 중요한 사실
- TP, SL, Size 조정으로 EV 부호는 바뀌지 않는다
- **EV는 조건이 결정한다**

### EV 분기 원인 (확정)
| 조건 | EV 상태 |
|:-----|:--------|
| MAE ≥ 10pt | 완전 붕괴 |
| MAE < 4pt | 폭발적 양수 |

**즉:**
> EV는 방향이 아니라 "얼마나 빨리 틀렸음을 인정했는가"로 결정된다.

- 하지만 EV, 안정성, 재현성은 avg ≥ 15에서만 확보

## 7. 왜 '완벽한 숏'은 구조적으로 어려운가

### 핵심 이유 3가지

**① 숏은 "확정 상태"가 아니라 "과정"이다**
- 롱: 수급이 쌓인다
- 숏: 수급이 끊긴다 → 끊김은 순간이 아니라 구간

**② 방출 이후는 확률이 급격히 열화된다**
- 방출 성공 → 급락 (짧고 명확)
- 방출 실패 → 흡수 → 노이즈
→ 시간이 지날수록 숏은 불리해진다

**③ 그래서 90%는 "국면 조건부"로만 존재한다**

| 구간 | WR |
|:-----|---:|
| 전반 BULL | 90.9% |
| 후반 BULL | 50.0% |
| 전체 안정 상한 | ~81% |

👉 **이건 실패가 아니라 물리적 상한**
