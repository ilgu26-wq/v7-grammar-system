# Raw Research Archive

This directory preserves **all original research artifacts**, experiments,
draft papers, and exploratory code exactly as they were produced.

Nothing in this directory is considered part of the immutable core system.

## Contents

- experiments/   : exploratory hypothesis tests and ablations
- paper/         : draft papers and research notes
- tests/         : experimental validation and simulation tests
- archive/       : deprecated or legacy engines
- main_pipeline.py : historical integration prototype

This archive exists for:
- auditability
- reproducibility
- research lineage preservation
